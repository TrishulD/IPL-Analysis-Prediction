# 🏏 IPL Cricket Analysis and Prediction System

## 📌 Project Description

This project presents a complete **IPL (Indian Premier League) data analysis and match prediction system** using Data Science and Machine Learning techniques. It explores historical IPL data to uncover patterns in team performance, player statistics, and match outcomes.

The system also builds a predictive model to forecast match winners based on key features.

---

## 🎯 Objectives

* Perform exploratory data analysis on IPL datasets
* Identify trends in team and player performance
* Visualize important cricket statistics
* Build a machine learning model to predict match winners

---

## 📊 Dataset

The project uses the following datasets:

* **matches.csv** → Contains match-level data (teams, winner, toss, venue, season)
* **deliveries.csv** → Contains ball-by-ball data (runs, batsman, bowler, dismissals)
